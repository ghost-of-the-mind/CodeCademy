Fonts and Text Styles

H1: Main page heading

  • Font-family: Libre Baskerville
  • Font-weight: 700
  • Font-size: 50px
  • Line-height: 1.5em
  • Letter-spacing: 0.1em
  • Text-transform: None

H2: Subheading

  • Font-family: Libre Baskerville
  • Font-weight: 700
  • Font-size: 40px
  • Line-height: 1.5em
  • Letter-spacing: 0.1em
  • Text-transform: None

P: Paragraph text

  • Font-family: Crimson Pro
  • Font-weight: 400
  • Font-size: 18px
  • Line-height: 40px
  • Letter-spacing: 0.1px
  • Text-transform: None

Fallback Fonts:

  • For Libre Baskerville: Georgia, sans-serif
  • For Crimson Pro: Georgia, serif
